## About
The example shows how to integrate [FastCRUD](https://github.com/benavlabs/fastcrud) with [SQLAlchemy](https://github.com/sqlalchemy/sqlalchemy) to create robust async CRUD APIs using [FastAPI](https://github.com/fastapi/fastapi).


## Running
```bash
# run API
fastapi dev api.py

# run tests
pytest -vx test_api.py
```
