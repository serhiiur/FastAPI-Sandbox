## About
This project provides a boilerplate CRUD API based on FastAPI and [FastCRUD](https://github.com/benavlabs/fastcrud) package, offering robust async CRUD operations and flexible endpoint creation utilities. Additionally the entire API is integrated with [SQLAdmin](https://github.com/aminalaee/sqladmin) package which provides an admin interface for managing API data.


## Running
```bash
fastapi dev api.py

# run tests
pytest -vx test_api.py
```
